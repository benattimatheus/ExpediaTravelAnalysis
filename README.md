Columns meaning

<ol>
<li>date_time: Timestamp</li>
<li>site_name: ID of the Expedia point of sale</li>
<li>posa_continent: ID of continent associated with site_name</li>
<li>user_location_country: ID of the country the customer is located</li>
<li>user_location_region: ID of the region the customer is located</li>
<li>user_location_city: ID of the city the customer is located</li>
<li>orig_destination_distance: Physical distance between hotel and customer at the time of search</li>
<li>user_id: Customer ID</li>
<li>is_mobile: 1 if the user connected from mobile, 0 otherwise</li>
<li>is_package: 1 if the click/booking was generated as part of a package, 0 otherwise</li>
<li>channel: ID of marketing channel</li>
<li>srch_ci: Checkin date</li>
<li>srch_co: Checkout date</li>
<li>srch_adults_cnt: Number of adults in the hotel room</li>
<li>srch_children_cnt: Number of children in the hotel room</li>
<li>srch_rm_cnt: Number of hotel rooms in the search</li>
<li>srch_destination_id: ID of the destination</li>
<li>srch_destination_type_id: Type of destination</li>
<li>is_booking: 1 if booking, 0 if click</li>
<li>cnt: Number of similar events in the context of the same user session</li>
<li>hotel_continent: Continent of the hotel</li>
<li>hotel_country: Country of the hotel</li>
<li>hotel_market: Hotel market</li>
<li>hotel_cluster: ID of a hotel cluster</li>
</ol>
